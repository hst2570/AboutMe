• [ AngularJS ]
    • MVC 모델을 지원하는 프론트 엔드 개발 프레임워크.
    • 생산성이 좋아진다.
    • 쉽고 강력한 기능이 많다.

      기능 예제보기

      • 입력한 결과를 바로 보여주는 기능.

      • 반복되는 데이터 처리

    • 참고 사이트

      • Angular 공식 사이트 ( https://angularjs.org )

      • Angular 튜토리얼

  • [ ReactJS ]
    • 프레임워크가 아니라 라이브러리이다.
    • 컨트롤러나 AJAX 기능이 아닌 뷰에 초점이 맞춰져있다.
    • virtual DOM을 사용한다. (이해 중)
    • JSX 기능이 가독성을 높여주고 쉽다.

    • 참고 사이트

      • React 공식 사이트 ( https://facebook.github.io/react/ )

      • Velopert님의 React강좌

      • Mozilla 자바스크립트 참고자료

  • [ Spring framework ]
    • Java 기반의 프레임 워크.
    • DI, AOP를 적극 수용했다.
    • 러닝커브가 높아 기술을 사용하는데 시간이 오래걸렸음.
    • 새로운 개발 디자인인 DDD 적용 공부 중.

    • 참고 사이트

      • Spring framework reference

      • JPA 레퍼런스 (번역)

      • 한국 스프링 사용자 모임
